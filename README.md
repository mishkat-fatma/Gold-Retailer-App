>Full-stack gold retailer management platform with live bullion rate integration, dynamic theming, margin configuration, and real-time pricing architecture.
>Features
-Live bullion pricing architecture using WebSocket integration
-Firebase authentication system
-Dynamic shop branding & theming
-Margin and purity configuration
-Freeze/unfreeze rate controls
-Real-time preview dashboard
-Responsive admin panel
-Web deployment using Vercel
>Tech Stack
-TypeScript
-React Native / Expo
-Firebase
-WebSocket
-Vercel
>Deployed Link: src="https://gold-retailer-app.vercel.app"
